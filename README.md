# Сайт: грузоперевозки Дмитрий Алдошин

Статический лендинг: одна страница, без сборки.

## Структура

- `index.html` — точка входа (удобно для GitHub Pages и обычного хостинга)
- `site_gazel_gruzoperevozki_husband.html` — та же разметка (можно открывать напрямую)
- `images/dmitry-aldoshin.webp` — портрет
- `images/truck-front.png`, `images/truck-rear.png` — фото машины (копии из файлов в корне)
- Исходные снимки в корне: `машина вид спереди.png`, `машина вид сзади.png`

## Как открыть локально

Откройте в браузере `index.html` двойным щелчком или через «Открыть файл». Пути к картинкам относительные — важно не разносить файлы по разным папкам без правки `src`.

## Что исправлено в вёрстке

- Закрыты лишние `</div>` и объединён список в блоке героя (раньше разметка была сломана).
- Кнопки Max и ВКонтакте ведут на реальные профили; в блоке контактов — текстовые ссылки с якорями `#max` и `#vk`.
